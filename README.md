# 🚀 NeuroForge — Responsive Landing Page

A modern, responsive landing page designed for an online course platform focused on generative AI learning.

## 📌 Project Overview

This project is a fully responsive landing page built using **HTML, CSS, and JavaScript**.
It focuses on clean design, structured layout, and smooth user experience.

The goal was to create a realistic product landing page that highlights courses, features, and user engagement.

---

## ✨ Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* 🎨 Clean and modern UI with consistent styling
* 📌 Smooth scrolling navigation
* 🍔 Mobile-friendly hamburger menu
* ⚡ Scroll-based reveal animations
* 🧩 Organized sections:

  * Hero section
  * Features
  * Courses
  * Testimonials
  * Call-to-action
  * Footer

---

## 🛠 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla JS)** – Interactivity

---

## 🎯 Key Focus Areas

* Responsive layout using Flexbox and Grid
* Clean and maintainable code structure
* User-friendly navigation and UI
* Realistic landing page design for a product
